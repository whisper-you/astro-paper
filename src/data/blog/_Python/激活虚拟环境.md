## 虚拟环境

### Python虚拟环境

#### 1. 创建虚拟环境

```bash
python -m venv 环境名称(.venv)
```

#### 2. 激活虚拟环境

```bash
source 环境名称/bin/activate
```

#### 3. 退出

```bash
deactivate
```

#### conda虚拟环境

```bash
conda create -n 环境名称(myenv) python=版本号
conda activate 环境名称
conda deactivate
```

### virtualenv虚拟环境

```bash
pip install virtualenv   #安装virtualenv
virtualenv 环境名称
virtualenv -p python版本号 环境名称
source 环境名称/bin/activate
deactivate
```
